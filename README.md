## vue-project
a vue starter for webpack 4.23 +
## 内容
- webpack 4+
- vue
- vue-router
- vuex
- 单文件组件
- splitChunk
- gzip
## 开始

``` bash
# clone this repo
git clone 
# Install project dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
